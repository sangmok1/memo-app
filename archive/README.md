# 완료 기록 아카이브

완료한 할일이 **메모별 · 날짜별**로 저장되는 폴더입니다.

## 폴더 구조

```
archive/
  20260707-k3m9/          ← 메모 ID (년월일-난수)
    2026-07-07.json
    2026-07-07.md
  20260708-x2a1/
    2026-07-08.json
    2026-07-08.md
```

- 메모를 **+** 버튼으로 만들 때 `archive/` 안에 해당 ID 폴더가 생성됩니다.
- 형식: `YYYYMMDD-xxxx` (KST 기준 날짜 + 4자리 난수)

## 파일 형식

- `YYYY-MM-DD.json` — 구조화된 완료 기록
- `YYYY-MM-DD.md` — 읽기 쉬운 마크다운

## 저장 시점

1. **날짜가 바뀔 때** — 해당 메모의 어제 완료 항목
2. **오늘 한일 정리하기** — 해당 메모의 오늘 완료 항목

⚙ **완료 기록 폴더 열기** → 현재 선택된 메모의 archive 폴더가 열립니다.
