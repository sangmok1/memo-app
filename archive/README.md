# 완료 기록 아카이브

완료한 할일이 날짜별로 저장되는 폴더입니다.

## 파일 형식

- `YYYY-MM-DD.json` — 날짜별 완료 기록 (프로그램용)
- `YYYY-MM-DD.md` — 같은 내용을 읽기 쉽게 정리한 마크다운

## 저장 시점

1. **날짜가 바뀔 때** — 어제 오늘 할일 중 완료된 항목 저장
2. **오늘 한일 정리하기** — 오늘 완료된 항목 저장

## 예시 (2026-07-07.json)

```json
{
  "date": "2026-07-07",
  "dateLabel": "2026년 7월 7일 (화)",
  "items": [
    { "text": "회의 준비", "savedAt": "2026-07-07T02:30:00.000Z" }
  ]
}
```

주간/월간 리뷰 시 이 폴더의 파일들을 날짜순으로 보면 됩니다.
